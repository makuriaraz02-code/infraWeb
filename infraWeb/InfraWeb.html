Bits & Bytes Notes

1. Bit (Binary Digit)
The smallest unit of data in a computer. Can only be 0 or 1. Represents one binary state: off/on, false/true, or low/high.

2. Byte
A group of 8 bits.
Example: 01000001 = 1 byte (represents the letter A in ASCII).
Bytes are used to measure file sizes and memory.
